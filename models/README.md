# Models Directory

Place your public ONNX model file here before running benchmarks.

Suggested lightweight options:

- `mobilenetv2-7.onnx` from the ONNX Model Zoo
- `resnet18-v1-7.onnx` from the ONNX Model Zoo

Example expected path:

- `models/mobilenetv2-7.onnx`

The project does not bundle a pretrained model by default so the repository stays lightweight and reproducible.
